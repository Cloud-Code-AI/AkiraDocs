# @akiradocs/core

Core functionality for Akiradocs - A modern documentation framework.

## Installation 

```bash
npm install @akiradocs/core
``` 

## Features

- Content Management
- Markdown Processing
- File Conversion Utilities
- Navigation Helpers
- TypeScript Support

## Usage

```typescript
typescript
import { getContentBySlug, parseMarkdown } from '@akiradocs/core'
// Get content by slug
const content = getContentBySlug('articles', 'my-article')
// Parse markdown content
const parsed = parseMarkdown(markdownContent)
```
